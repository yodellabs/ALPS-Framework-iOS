Delegate method that should be used for AR applications. Supplies the current location in the ALPS coordinate system and the transform from ARKit's reference frame to ALPS's reference frame.

Parameters
managerPointer to the location manager that sent a new location.
locationThe location in the ALPS reference frame with accuracies in meters.
poseTransformThe pose transform to translate and rotate the ARKit reference from to the ALPS reference frame.
rangeDataPointsNSArray containing ALPSRangeDataPoints from received ultrasound transmissions. These each contain a beacon MAC, time slot and range from the phone to the beacon.
timeStampUnix time of when the data was received.
